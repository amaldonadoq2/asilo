## Instalacion

```sh

 git clone https://github.com/amaldonadoq2/asilo.git


 cd asilo

 cp .env.example .env

 composer install


 php artisan key:generate


 php artisan migrate


 php artisan passport:install

 npm install


 npm run dev # o npm run watch


 npm run production

 php artisan serve

```
